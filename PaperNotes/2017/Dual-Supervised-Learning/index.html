Dual Supervised Learning

Posted on 2018-02-13 | In Paper Notes | | Views:

Paper Link

前言

很多時候，某個任務的input/output是另一個任務的output/input，像是下面的dual form tasks，儘管這些任務是雙向對稱的，但大多各自訓練model，而這篇研究目標是利用此雙向特性達到更好的效果。

  • Supervised learning tasks in dual forms
    • translation (Fr -> En / En -> Fr)
    • speech recognition / text to speech
    • image classification / image generation

內容

目的：同時訓練 dual models

舉例證明效果更好：

  1. Neural Machine Translation
  2. Image Processing on CIFAR-10
  3. Sentiment Analysis on IMDB dataset
Def Illustration Probability
primal task: x $\rightarrow$ y P(y\ x; $\theta$~xy~)
dual task: y $\rightarrow$ x P(x\ y; $\theta$~yx~)

Probabilistic Duality:

$$P(x, y) = P(x)P(y|x; \theta_{xy}) = P(y)P(x|y; \theta_{yx}) \
\theta: model$$

困難：如果個別計算 cost function ，則上方公式難保成立。

object 1: $min_{\theta_{xy}}(1/n)\sum^n_{i=1}\ell_1(f(x_i;\theta_{xy}), y_i),$
object 2: $min_{\theta_{yx}}(1/n)\sum^n_{i=1}\ell_2(g(y_i;\theta_{yx}), x_i),$
s.t. $P(x)P(y|x;\theta_{xy}), \forall x,y,$

解決：將上述公式轉為 penalty term ( 使用 Lagrange Multipliers) ，可將penalty term 視為 data-dependent regularization term.

然而，因為$P(x), P(y)$ (marginal distributions) 並不好取得，用$\hat{P}(x), \hat{P}(y)$ (empirical marginal distributions) 取代。

利用 Lagrange multipliers 和 equality constraint of probabilistic duality(以下稱雙向機率對等條件) 代入上方 objective functions.

  1. 將雙向機率對等條件 regularize
    $$\ell_{duality}=(log\hat{P}(x)+logP(y|x;\theta_{xy})-log\hat{P}(y)-logP(x|y;\theta_{yx}))^2$$
  2. Minimize the weighted combination and the above regularization term，針對 objective 1. & 2. 加上 $\lambda\ell_{duality}$，可用SGD, Adam, Adadelta。

備註：提供 theoretical analysis 確保DSL的 generalization bound；提供演算法。

討論

此方法也對 unsupervised learning 帶來好處，以翻譯為例，$x \rightarrow y’$，再由$y’ \rightarrow x’$，透過減少$x$和$x’$的差異，雙向models都會被優化。

$\ell_{duality}$ (regularization term) 是 data dependent，意指每個樣本影響 regularization term，每個 model 影響其他 model 的 regularization；然而，Lasso, SVM 是 data independent。

DSL 之必要條件：

  1. Duality 必須存在(for the two tasks)
  2. Primal 和 Dual models 必須可訓練(trainable)
  3. $\hat{P}(X)$ and $\hat{P}(Y)$ 必須可取得

應用

機器翻譯

  • 參考 Bahdanau et al., 2015; Jean et al., 2015
  • Tasks
    1. En $\leftrightarrow$ Fr
      • Dataset from previous papers
    2. En $\leftrightarrow$ De(Germany)
      • Dataset from newstest 2012, 2013, 2014
    3. En $\leftrightarrow$ Zh(Chinese)
      • Dataset from NIST2006, 2008, 2012
  • Model
    • 使用 GRU as the recurrent module (上述 paper) 實踐 seq2seq model
    • Word embedding 維度為620
    • Hidden node 為 1000
    • 單字數量分別為 30k, 50k, 30k
    • Baseline: RNNSearch
  • Evaluation Metrics
    • Tokenized case-sensitive BLEU
    • 分數越大品質越好
    • case-insensitive BLEU for Zh $\rightarrow$ En
  • Training Procedure
    1. Initialize two models by two warm-start models（上述 paper），從後段得知為 RNNSearch 訓練出來的 models 作為 initial
    2. Use SGD with minibatch size of 80 to optimize
    3. Initialize learning rate = 0.2, 持續減半直到 BLEU score on the validation set 無法明顯成長
    4. The gradient clip is set as 1.0, 5.0 and 1.0 individually
    5. 根據經驗，$\lambda_{xy}$ and $\lambda_{yx}$ are set as 0.01
  • Result
    • 可以同時改良兩個models
    • En $\rightarrow$ Fr 進步最多
    • Regularize probabilistic duality constraint 會降低 $\ell_{duality}$，RNNSearch $\rightarrow$ DSL
  • Improvement
    • Replace “UNK”(unknown word) with corresponding word-level translations
    • 使用更好的 models 作為 initials ，取代原本從 RNNSearch 得到的 warm-start models
  • Advantages
    • Improve by leveraging the stuctural duality of sentences
    • consider more contextual information
    • handle the plural form

圖像處理

  • Dataset: CIFAR-10 with 10 classes of images
  • ResNet for image classifiaciont; PixelCNN++ for image generation
  • $\chi$ for image space; $y$ for category space related to CIFAR-10
  • Marginal Distributions use uniform distribution
  • Models
    • Baseline for image classification: ResNet-32, ResNet-110; for image generation, PixelCNN++
  • Evaluation Metrics
    • classification error rates for image classfication.
    • bits per dimension(bpd) (Salimans et al., 2016) for image generation. for an image x with label y, the bpd is
      $$ -(\sum^{N_x}_{i=1}logP(x_i|x_{x<i}, y)) / (N_xlog(2)) $$
  • Training Procedure
    • Initialize both the primal and dual models with ResNet model and PixelCNN++ model pre-trained independently and separately.
    • Initialize learning rate of image classification model as 0.1 and that of image generation model as 0.0005
    • 將近兩週才收斂
  • Result
    • Image classification 有進步好棒棒
    • Image generation 比較困難，採用 ResNet-110 則 bpd 減少0.01，0.01不大卻是進步許多

情感分析

  • $\chi$ 代表句子，$y$代表情緒
  • Dataset IMDB movie review
  • Marginal Distributions use uniform distribution, positive(0.5) or negative(0.5)
  • Model
    • LSTM for sentiment classification model
    • embedding dimension = 500
    • hidden layer size = 1024
    • LSTM for sentence generation, input:
      $$W^e_wE_wx_{t-1} + W^e_sE_sy$$
      $x_{t-1}$ t-1’th word
      $E_w$ and $E_s$ embedding matrices and sentiment label
      $W’s$ represent the connections between embedding matrix and LSTM cells
    • 一個字一個字產生成句子，該字被產生的機率和下述公式成正比，contextual language model(CLM) $$exp(W^d_wE_wx_{t-1}+W^d_sE_sy+W_hh_{t-1})$$
  • Evaluation Metrics
    • Use error rate
  • Training Procedure
    1. Baseline models: use Adadelta as the optimization method to train model
    2. Initialize the two models for DSL
    3. Use plain SGD
    4. Initialize learning rate of 0.2
    5. Decrease it to 0.02 當 validation set 沒明顯進步時
    6. $\lambda_{xy}=(5/l_x)^2$ and $\lambda_{yx}=(0.5/l_x)^2$，$l_x$ 為x的長度
  • Result
    • 成功減少分類錯誤
    • 稍微改進 perplexity for sentence generation，原因可能為 label 僅能提供 1 bit 資訊
    • Baseline model 通常抓 training data 的高頻字
    • CLM in DSL 可以抓到更準確或強烈的字

探討

  • 根據三項測驗，發現不需要做 warm-start models 也可以達到比 baseline 好的效果，差別只在速度
